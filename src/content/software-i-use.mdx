# Essentials

- **Kitty** - terminal emulator
- **AltTab** - alt tab with window previews
- **AeroSpace** - Tiling WM
- **Homerow** - Keybinds for clicking on stuff
- **LinearMouse**
- **Sketchybar**
- **Affinity Suite**
- **Shottr** - screenshots
- **Pixelmator pro**
- **Supercharge** - various mac fixes
- Raycast
- [System Color Picker](https://apps.apple.com/us/app/system-color-picker/id1545870783?mt=12)
- **Screen studio** - smooth screen recordings
- **Karabiner** - rebinds
- **Loop/FlashSpace** - manual window management
- **MediaMate** - fix volume/brightness popup
- [Folder Preview](https://apps.apple.com/us/app/folder-preview/id6698876601?mt=12)
- **CompressX** - pretty good ffmpeg wrapper with nice ui
- **motrix** - torrent client
- **Displaperture** - round display corners
- **IINA** - good video player
- **Shareful** - more useful options in share menu
